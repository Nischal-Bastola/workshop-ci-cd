What is this Workshop?

This intensive 5-day program covers the full foundation of cloud computing with a strong focus on Amazon Web Services. You'll move from core cloud theory to hands-on AWS labs — deploying servers, configuring storage, setting up databases, designing secure networks, and understanding the DevOps lifecycle that powers modern software delivery.

Each day builds on the last, giving you a coherent, practical skill set rather than isolated knowledge snippets. By the end, you'll confidently navigate the AWS console and understand how real production architectures are built.

🎯

Curriculum

Workshop Syllabus

A structured 5-day curriculum. Days 1–3 are detailed below.

Day 01

Cloud & AWS Foundations

  • Cloud Fundamentals

    Understanding cloud computing models (IaaS, PaaS, SaaS), deployment types (public, private, hybrid), and core benefits of moving to the cloud.

  • Cloud Ecosystem & AWS Introduction

    Overview of the major cloud providers, AWS global infrastructure, regions, availability zones, and the AWS Management Console.

  • AWS Basics & IAM

    Setting up an AWS account, IAM users, groups, roles, and policies. Security best practices and the principle of least privilege.

Cloud Theory IAM AWS Console
Day 02

Compute, Storage & Databases

  • Amazon EC2 (Compute)

    Launching and managing EC2 instances, instance types, AMIs, key pairs, security groups, and connecting via SSH. Lab: deploy a web server.

  • Amazon S3 (Storage)

    Buckets, objects, storage classes, versioning, static website hosting, and access control with bucket policies and ACLs.

  • Database Fundamentals

    SQL vs NoSQL trade-offs, Amazon RDS setup and multi-AZ deployment, and Amazon DynamoDB for key-value workloads.

EC2 S3 RDS DynamoDB
Day 03

Networking, Security & DevOps

  • Networking Basics

    IP addressing, CIDR notation, subnets, routing tables, and how traffic flows through a network — the prerequisite for everything AWS networking.

  • VPC Components & Security

    Building a custom VPC: public & private subnets, Internet Gateway, NAT Gateway, Security Groups vs NACLs, and VPC Peering.

  • DevOps Fundamentals & Lifecycle

    Introduction to DevOps culture, the CI/CD pipeline, Infrastructure as Code concepts, and how AWS tools (CodePipeline, CloudFormation) fit in.

VPC Networking CI/CD
Day 04 Advanced AWS Services & Serverless Lambda · API Gateway · CloudWatch Day 05 Capstone Project & Certification Prep Full Architecture · Mock Exam
